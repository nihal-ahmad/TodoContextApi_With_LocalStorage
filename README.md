## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

- Live Link : https://persistent-todo.netlify.app

## Features

- Add Todo
- Edit Todo
- Delete Todo
- Persistent Todo
